# ELEVENS Restaurant Website

> 🍽️ **Premium Fine Dining Experience** - A luxury restaurant website built with React, TypeScript, Tailwind CSS, and Framer Motion

## ✨ Features

- 🎨 **Premium Glassmorphism Design** - Modern luxury aesthetics
- 🌙 **Dark Theme** - Elegant color palette (#111111, #D4AF37, #F8F4EE)
- ⚡ **Smooth Animations** - Framer Motion powered transitions
- 📱 **Fully Responsive** - Mobile-first design approach
- ♿ **Accessible** - WCAG 2.1 AA compliant
- 🔍 **SEO Optimized** - Complete meta tags and structured data
- 🎯 **Performance Optimized** - Fast loading and smooth interactions

## 🏗️ Tech Stack

- **Frontend Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Build Tool:** Vite
- **Fonts:** Playfair Display (headings), Poppins (body)

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Sections

1. **Hero** - Premium landing with animated elements
2. **About** - Restaurant story and statistics
3. **Why Choose Us** - Feature highlights
4. **Menu** - Interactive food menu with filtering
5. **Reviews** - Customer testimonials
6. **Gallery** - Image showcase
7. **Contact** - Reservation form and contact info
